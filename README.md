## 🌟 Overview

SkillSphere is a production-ready online learning platform built with Next.js 14+, designed to deliver a smooth and engaging user experience for learners.

With secure authentication, responsive UI, and modern animations, SkillSphere focuses on simplicity, performance, and scalability — making it ideal for real-world applications and portfolio showcase.

## 🚀 Getting Started

### Prerequisites

* Node.js 18+
* MongoDB (local or Atlas)
* Google OAuth credentials

---

### Installation

```bash
git clone https://github.com/Abdhullah-Al-Maruf/skillSphere-online-platform.git
cd skillSphere-online-platform
npm install
```

---

### Environment Variables

Create `.env.local`:

```env
MONGODB_URI=your_mongodb_uri

BETTER_AUTH_SECRET=your_secret
BETTER_AUTH_URL=http://localhost:3000

GOOGLE_CLIENT_ID=your_client_id
GOOGLE_CLIENT_SECRET=your_client_secret
```

---

### Run Locally

```bash
npm run dev
```

Open http://localhost:3000 🚀
